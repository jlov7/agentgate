# Project Independence Statement

## Overview

AgentGate is an independent technical research project developed by practitioners in PwC's AI Risk practice. It serves as a reference implementation demonstrating containment-first security controls for AI agents.

## Relationship to PwC

- This project was developed as independent technical research
- The concepts and implementation represent practitioner perspectives, not official firm positions
- PwC has not independently verified the code for production use
- No client data or proprietary information was used in development

## Scope and Limitations

AgentGate is a **reference implementation** intended to:
- Demonstrate practical approaches to agent containment
- Support academic and policy discussions on AI agent security
- Provide a starting point for organizations building production controls

It is **not**:
- Production-ready infrastructure
- A commercial product or service offering
- Certified or compliant with any regulatory framework
- A substitute for professional security assessment

This project is released under the Apache License 2.0. The concepts described align with emerging industry best practices and do not represent proprietary PwC methodologies.
