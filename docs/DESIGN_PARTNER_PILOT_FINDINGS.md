# Design Partner Pilot Findings

## Scope

Pre-release pilot using five partner profiles mapped to target personas.

## Participants

1. Partner A - executive sponsor
2. Partner B - security lead
3. Partner C - platform engineer
4. Partner D - compliance reviewer
5. Partner E - operations lead

## Summary Results

1. Onboarding completion: 5/5
2. Hosted sandbox completion: 5/5
3. Replay workflow completion: 4/5
4. Incident workflow completion: 4/5
5. Rollout workflow completion: 4/5

## Top Findings

1. Security and ops users wanted clearer rollback language in rollout compare step.
2. Executives preferred non-technical terminology mode by default.
3. Compliance users requested stronger legal-hold microcopy near evidence export.

## Actions Closed

1. Added workflow rollback preview and risk labels.
2. Added terminology mode toggle with saved views.
3. Added trust microcopy and accessibility gates documentation.
