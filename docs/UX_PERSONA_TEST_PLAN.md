# UX Persona Test Plan

## Personas Under Test

1. Executive Sponsor
2. Security Lead
3. Platform Engineer
4. Compliance Reviewer

## Test Scenarios

1. First-time onboarding path completion.
2. Trial completion and transcript/evidence export.
3. Replay -> rollout decision flow.
4. Incident quarantine -> release flow.
5. Trust posture review and support escalation lookup.

## Success Criteria

1. Persona reaches intended goal without facilitator intervention.
2. No dead ends in navigation.
3. Error states point to corrective actions.
4. User can explain what happened and what to do next.
