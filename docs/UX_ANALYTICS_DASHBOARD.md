# UX Analytics Dashboard

This dashboard summarizes north-star metrics, friction signals, experiments, diagnostics, and feedback captured from the interactive UX surfaces.

## North-star Metrics

Track and improve:

1. `time_to_value`
2. `activation`
3. `journey_completion`
4. `drop_off`
5. `confidence_score`

<div id="ag-ux-analytics" class="ag-lab"></div>

## Event Model

Interactive workflows emit `ag-ux-event` records for onboarding steps, flow outcomes, backtracks, gate blocks, and summary generation.

## Usage

1. Run journeys in hosted sandbox, demo lab, replay, incident, and rollout pages.
2. Open this dashboard to inspect event-derived metrics.
3. Use weekly scorecard and diagnostics to prioritize UX fixes.
