# UX Phase Gates

## Alpha UX Gate

Must pass:

1. Journey-first IA is live (`Start Here`, `Journey Map`, grouped navigation).
2. Core onboarding checklist persists and resume works.
3. Hosted sandbox and demo lab provide guided next-step transitions.

## Beta UX Gate

Must pass:

1. Persona workspace patterns are documented and routeable.
2. Content style guide and glossary are adopted for primary pages.
3. Accessibility smoke checks pass for key journey surfaces.
4. UX tracker has >50% item closure with evidence links.

## Release UX Gate

Must pass:

1. End-to-end persona journey test plan executed.
2. Phase metrics reported (activation, completion, friction).
3. Launch playbook and post-launch monitoring runbook are complete.
4. Open UX items are non-blocking and explicitly accepted.
