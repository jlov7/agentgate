# Journey Map

<div data-ag-context></div>

Use these role-based journeys to move from exploration to safe production operation.

## Journey 1: Evaluate Quickly (POC)

1. Open <a href="../HOSTED_SANDBOX/">Hosted Sandbox</a> and run all seeded flows.
2. Run <a href="../DEMO_LAB/">Interactive Demo Lab</a> for scenario narrative and blast-radius context.
3. Generate trial artifacts in <a href="../TRY_NOW/">Try in 5 Minutes</a>.
4. Share outputs with stakeholders using <a href="../DEMO_DAY/">Demo Day Playbook</a>.

## Journey 2: Integrate Safely (Engineering)

1. Validate health and tools API behavior in hosted sandbox.
2. Validate policy behavior with <a href="../REPLAY_LAB/">Replay Lab</a> before rollout.
3. Configure rollouts in <a href="../TENANT_ROLLOUTS/">Tenant Rollouts</a>.
4. Confirm observability via <a href="../OBSERVABILITY_PACK/">Observability Pack</a>.

## Journey 3: Operate with Confidence (Ops/Security)

1. Monitor risk posture and thresholds.
2. Trigger and release incidents via <a href="../INCIDENT_RESPONSE/">Incident Response</a>.
3. Validate evidence and support bundles from <a href="../OPERATIONAL_TRUST_LAYER/">Operational Trust Layer</a>.
4. Rehearse rollback flow for tenant canaries.

## Journey 4: Prove Compliance (Audit)

1. Export signed evidence (JSON/HTML/PDF).
2. Map controls via <a href="../COMPLIANCE_MAPPINGS/">Compliance Mappings</a>.
3. Validate SLA/SLO and support posture in trust docs.
4. Package reproducible artifacts for review.
