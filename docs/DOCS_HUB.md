## Key Concepts

| Term | Definition |
|------|------------|
| **Policy Gate** | Rules that determine if an action is allowed, denied, or requires approval |
| **Kill Switch** | Emergency controls to instantly stop agents at session, tool, or global level |
| **Evidence Pack** | Tamper-proof audit records with cryptographic integrity verification |
| **Trace Store** | Append-only database of all agent actions |
| **MCP** | Model Context Protocol — the interface AI agents use to access tools |
