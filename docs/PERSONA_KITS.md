# Persona Kits

Persona kits provide preloaded scenario routes and expected artifacts for guided onboarding.

## Available Kits

- `docs/lab/personas/executive.json`
- `docs/lab/personas/security.json`
- `docs/lab/personas/engineering.json`
- `docs/lab/personas/compliance.json`

Use these kits to tailor demos, onboarding checklists, and first-week workspace defaults.
