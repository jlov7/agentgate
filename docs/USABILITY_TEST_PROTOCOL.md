# Usability Test Protocol

## Goal

Validate that each persona can complete critical journeys with low friction and high confidence.

## Test Modes

1. Moderated sessions (30 minutes): facilitator observes decision-making and confusion points.
2. Unmoderated sessions (10-15 minutes): scripted tasks with telemetry capture and follow-up survey.

## Participant Mix

1. Executive sponsor (non-technical)
2. Security lead
3. Platform engineer
4. Compliance reviewer
5. Ops/on-call engineer

## Core Tasks

1. Complete onboarding path to first value.
2. Run hosted sandbox and export trial report.
3. Execute replay workflow and generate a regression patch.
4. Perform incident quarantine decision and summary export.
5. Run staged rollout flow and identify rollback trigger.

## Success Rubric

1. Task completion without external intervention.
2. Time-on-task within target bounds.
3. Correct interpretation of risk and next action.
4. Reported confidence score >= 4/5.
