# UX Glossary

| Term | Meaning |
|---|---|
| First Value | The first meaningful successful outcome a new user achieves (for AgentGate: successful flow + evidence). |
| Journey | A sequenced path through related tasks to reach an operational goal. |
| Persona | A role-based user profile with distinct goals and language needs. |
| Progressive Disclosure | Revealing advanced options only when relevant, instead of showing everything at once. |
| Blast Radius | The scope of impact from a policy or operational change. |
| Evidence Pack | Exported structured proof artifacts (JSON/HTML/PDF) showing what happened and why. |
| Canary Rollout | Gradual deployment to a subset before full rollout. |
| Quarantine | Session containment action triggered by risk thresholds. |
| Runbook | A step-by-step operational response procedure. |
